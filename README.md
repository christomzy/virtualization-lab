# Virtualization Lab – Hyper-V, Citrix XenServer & VMware ESXi

> A hands-on virtualization project built as part of the **Cloud & Infrastructure Specialist** program.  
> Three hypervisor platforms were installed, configured, and compared in a nested virtualization environment.

---

## Overview

This project covers the installation, configuration, and administration of three industry-standard virtualization platforms:

| Platform | Key Features Demonstrated |
|---|---|
| [Microsoft Hyper-V](#hyper-v) | Nested virtualization, Live Migration, Active Directory, Kerberos delegation |
| [Citrix XenServer](#citrix-xenserver) | XenMotion (live migration), snapshots, Storage Repositories, XenCenter |
| [VMware ESXi](#vmware-esxi) | vSwitches, Port Groups, VMFS datastores, snapshots, VMware Host Client |

Each platform was documented step by step with screenshots, technical explanations, and justifications for configuration choices.
